Getting started

Install via NPM...

$ npm install jml

... or add script to your web page.

<script type="text/javascript" src="/path/to/jml.js"></script>

Simple usage

jml.render([['div.container',
  ['a.link', {_href: 'http://tenphi.com'}, 'Link text'],
  ['img.image', {_src: '/myphoto.jpg', _alt: '', width: '100px'}]
]]);

output (with beautifying):

<div class="container">
  <a class="link" href="http://tenphi.com">Link text</a>
  <img class="image" src="/myphoto.jpg" alt="" style="width: 100px;" />
</div>

Render container

Parameters order is strict:

  • tag name
  • id
  • classes

Example: ['a#id.class.second-class']

Alternative declaration

Declaration of tag properties can be object:

jml.render([[{
  tag: 'a',
  id: 'link',
  classes: 'link-class'
}, {
  _href: 'http://tenphi.com'
}, 'Link']]);

Options for render

Render function can accept options:

var link = [function() {
  return ['a', {_href: this.url}].concat(this.body);
}];
var options = {
  url: 'http://tenphi.com',
  body: 'Link Text'
};
jml.render([link], options);

Advanced usage with Templates

Add template:

jml.view('Link', ['a', {_href: 'url'}, 'Link Text']);

then render it:

jml.render([['div.container', ['Link']]]);

Template options

Rewrite template:

jml.view('Link', [function() {
  return ['a', {_href: this.url}].concat(this.body);
}]);

and render it with options:

jml.render([['div.container', ['Link', {
  url: 'http://tenphi.com',
  body: 'Link Text'
}]]]);

JML Tools

There are several function-tools for JML:

  • jml.prop(name) - return property name 
var template = [['a', {_href: 'http://tenphi.com'}, jml.prop('content')]];
jml.render(template, {content: 'Link Text'});
  • jml.map(name, handler) - return result of map function on property name 
var head = [['head',
  ['title', 'Title of Site'], 
  jml.map('styles', function(file) {
    return [['link', {
      _rel: 'stylesheet',
      _type: 'text/css',
      _href: file
    }]];
  })
]];
jml.render(head, {styles: ['reset.css', 'main.css']});
  • jml.filter(name1, name2, ...) - return options object without properties that not listed
var template = [['div', jml.filter('width', 'height'), jml.prop('content')]];
jml.render(template, {
  width: '100px',
  height: '100px',
  content: 'Content Text'
});
